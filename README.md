# OSU Energy Analysis

## Overview

This project analyzes smart meter energy data, building metadata, and weather data to identify energy consumption patterns, evaluate building efficiency, and explore sustainability opportunities across Ohio State University facilities.

This repository follows a structured data engineering and analytics workflow built using Databricks and PySpark.

---

## Project Structure

The workflow is organized into four main notebooks:
<ol>
<li>01_load_and_join</li>
<li>02_energy_trends</li>
<li>03_efficiency_analysis</li>
<li>04_dashboard_tables</li>
</ol>
